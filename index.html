Why I'm Learning to Code

One year ago, in grade 10, I had the choice between taking grade 11 computer science, or taking grade 11 functions. I decided to drop my computer science class and get ahead in my math courses. Although I do not regret this choice, it sometimes leaves me with a sense of regret. I've always thought coding was the coolest thing, and now that I dropped the class, I thought I wouldn't be able to learn it anymore. This was a terrible mentality! I'm now realizing I don't need a class in school to teach me how to code, I can learn it myself. Starting off with HTML basics, I then learned CSS basics, and I can't wait to then learn JS basics. This basic programming set will then be completed with basic python. My plan is to then build on my front-end coding and develop my skills using modern web dev tools such as React and Tailwind. I can't wait to continue to grow and learn through programming!
